1.String

In JavaScript, a string is a primitive data type that represents a sequence of characters. You can create a string by enclosing text in single (' '), double (" "), or backtick ( ) quotes.

2.Number

In JavaScript, a number variable is used to store numerical values. Numbers in JavaScript are represented as a single data type, and you can use variables to store and manipulate them in your code

3.Boolean

A boolean variable in JavaScript is a variable that can only have two possible values: true or false. Booleans are often used for logical operations and conditional statements.

4.Undefined

In JavaScript, undefined is a special value and also a data type. If a variable is declared but not assigned a value, it is automatically assigned the value undefined.

5.Null

In JavaScript, null is a special value that represents the intentional absence of any object value. It is often used to explicitly indicate that a variable has no assigned value or that an object property does not point to any valid object.
